    Callbacks

    Callbacks are the core building blocks of cksp scripts. They define how your script reacts to various events in Kontakt, such as MIDI input, UI interactions, timers, and asynchronous operations. Each callback is triggered by specific events and runs to completion before returning control to Kontakt’s engine.

    Initialization and Persistence

    on init

    Runs once after the script compiles successfully (applying the script, loading an instrument, or restarting the audio engine). Kontakt allocates resources at this point. Just like when declaring variables or constructs outside of callbacks, declarations here are visible in any other callback or function. All UI control declarations live here (or in the global scope). cksp treats variable declarations (not ui control declarations) in on init as global, even if you are not speficially using the global keyword after declare. To declare a variable as local to on init, you can use the local keyword. In this case, this variable will not be accessible outside of on init and will not be preserved across callbacks.

    Declare a button
    1
2
3
4
    on init
    declare ui_button btn_do_sth
    btn_do_sth -> text := "Do something!"
end on

    on persistence_changed

    Fires right after on init or when a snapshot loads. Use it to restore state, re-label controls, or recompute values without re-running the full initialization logic from on init. To control whether only this callback is executed upon snapshot load use the builtin command set_snapshot_type(1).

function update_menu()
    if ($Sync = 1)
        move_control($Time, 2, 1)
    else
        move_control($Time, 0, 0)
    end if
end function

on persistence_changed
    call update_menu()
end on

on ui_control ($Sync)
    call update_menu()
end on

    pgs_create_key(NEXT_KEY, 128)
    declare ui_knob $First (0, 100, 1)

on pgs_changed
    if (pgs_key_exists(FIRST_KEY) and pgs_key_exists(NEXT_KEY))
        $First := pgs_get_key_val(FIRST_KEY, 0)
        %Next[0] := pgs_get_key_val(NEXT_KEY, 0)
        %Next[127] := pgs_get_key_val(NEXT_KEY, 127)
    end if
end on

    MIDI Event Callbacks

    on note

    struct Note
    pitch: int
    velocity: int

    function __repr__(self): string
        return "Note " & self.pitch & " velocity " & self.velocity
    end function
end struct

on note
    declare note := new Note(EVENT_NOTE, EVENT_VELOCITY)
    message(note)

    on controller

    Handles MIDI CC, Pitch Bend, and Channel Pressure. Built-ins: %CC[], $CC_NUM, $VCC_PITCH_BEND, $VCC_MONO_AT. Each incoming controller value updates %CC before the callback runs.

    on controller
    if (in_range($CC_NUM, 0, 127))
        message("CC " & $CC_NUM & " = " & %CC[$CC_NUM])
    else
        if ($CC_NUM = $VCC_PITCH_BEND)
            message("Pitch Bend = " & %CC[$CC_NUM])
        end if
        if ($CC_NUM = $VCC_MONO_AT)
            message("Channel Pressure = " & %CC[$CC_NUM])
        end if
    end if
end on

    on init
    declare const $NUM_PARAMS := 2
    declare %params[$NUM_PARAMS] := (0, 0)
    declare ui_slider $Cut (0, 1000000)
    declare ui_slider $Res (0, 1000000)
    declare %pidx_to_uid[$NUM_PARAMS]
    %pidx_to_uid[0] := get_ui_id($Cut)
    %pidx_to_uid[1] := get_ui_id($Res)
    set_control_par(%pidx_to_uid[0], $CONTROL_PAR_CUSTOM_ID, 0)
    set_control_par(%pidx_to_uid[1], $CONTROL_PAR_CUSTOM_ID, 1)
end on

function set_param()
    select (get_control_par($NI_UI_ID, $CONTROL_PAR_CUSTOM_ID))
        case 0
            set_engine_par($ENGINE_PAR_CUTOFF, %params[0], 0, 0, -1)
        case 1
            set_engine_par($ENGINE_PAR_RESONANCE, %params[1], 0, 0, -1)
    end select
end function

on ui_controls
    declare $idx := get_control_par($NI_UI_ID, $CONTROL_PAR_CUSTOM_ID)
    %params[$idx] := get_control_par($NI_UI_ID, $CONTROL_PAR_VALUE)
    call set_param()
end on

    set_knob_label($Volume, get_engine_par_disp($ENGINE_PAR_VOLUME, -1, -1, -1))
end on

    on init
    declare ui_knob $Tick (0, 99, 1)
    declare $tick_counter
    declare $direction
    set_listener($NI_SIGNAL_TIMER_MS, 10000)
end on

on listener
    if ($NI_SIGNAL_TYPE = $NI_SIGNAL_TIMER_MS)
        if ($direction = 0)
            inc($tick_counter)
        else
            dec($tick_counter)
        end if
        $Tick := $tick_counter
        if ($tick_counter = 99)
            $direction := 1
        end if
        if ($tick_counter = 0)
            $direction := 0
        end if
    end if
end on

    Runs after an asynchronous command finishes (load/save operations, MIR commands, etc.). $NI_ASYNC_ID identifies the finished command; $NI_ASYNC_EXIT_STATUS is 1 on success, 0 otherwise.

    on init
    declare $load_midi_file_id
    declare ui_button $Load
end on

on ui_control ($Load)
    $load_midi_file_id := load_midi_file("path/to/file.mid")
    while ($load_midi_file_id # -1)
        wait(1)
    end while
    message("MIDI file loaded!")
end on

on async_complete
    if ($NI_ASYNC_ID = $load_midi_file_id)
        $load_midi_file_id := -1
        if ($NI_ASYNC_EXIT_STATUS = 1)
            message("Async load succeeded")
        else
            message("Async load failed")
        end if
    end if
end on
